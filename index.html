Summary: Lattitude vs x

In this project we sought to determine the connections between lattitude and cloudiness, humidity, wind speed and maximum temperature. To accomplish this analysis OpenWeatherMap API was used to create a dataset using a randomized list of 500 cities. After assembling the dataset, Matplotlib was used to plot the various aspects of weather listed above vs. latitude. Here you will find the source data and visualizations created as part of the project along with explanations and descriptions of trends and correlations.
